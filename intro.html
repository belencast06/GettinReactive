Introduction to using React

Step 1: Open Project Folder

  1. 1. Open your code editor (VS Code)
    File → Open → Find the folder you just created during the Pre-Workshop Installation Guide

Step 2: Observation over React

  1. Open the index.html file
    Inside the public folder, open the index.html file.

React is called a single-page application. React loads
the entire app on this index.html page.
  2. Locate the root div
    Find the line <div id="root"></div> — this is where your React app will render.
  3. Open the src/index.js file
    Look for the line:
const root = ReactDOM.createRoot(document.getElementById('root'));
  4. Understand root rendering
    This line tells the browser to render your entire app inside the root div.
    Next lines use root.render(...), which mounts your App component to the screen.
  5. Open App.js
    Inside src/App.js, find the return function. App.js returns HTML to index.html.
    React is a structured way to return HTML and CSS through components.
  6. Launch the app
    In VS Code, open a new terminal and run:
npm start
  7. Add a header
    Inside App.js, within the return function, add:
<header>My First React App</header>
  8. Add an image
    Below the header, add:
<img src="IMAGE_URL" alt="description" style={{width: "100%", maxWidth: "600px", height: "auto"}} />
  9. >Edit App.css
    Open src/App.css — this is where you define your styles.
  10. Change the background color
    In the .App-header class, replace the default background color with your own.
  11. Complete your challenge
    Pick a challenge and complete it by the end of the workshop.
Each challenge increases in difficulty.
